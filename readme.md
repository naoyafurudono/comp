## ズレるところ

- pprint を作ってパーサが合ってるか確認

    困ったり迷ったら参考にみに行くけど、デバッグは自力でしないといけない。
  - `make print ARGS="program"` で program にカッコをつけて表示する
  - lexer 向けにも作りかけているけど、デバッグに欲しくならなかったので中途半端
